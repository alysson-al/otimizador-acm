Passo a passo para gerar sua chave gratuita:

  1. Acesse o Google AI Studio e faça login.
  2. Clique no botão azul "Create API key".
  3. Na janela que abrir, clique no botão cinza "Create API key in new project".
  4. Copie a chave gerada (ela começa com AIzaSy...) e cole no campo abaixo.

A IA é usada apenas quando você importa um PDF. Para CSV, ela não é necessária. A chave fica salva localmente.
